# 🛡️ PhishGuard

A Flask-based cybersecurity project that detects phishing emails and suspicious URLs while improving cybersecurity awareness.

---

## 🚀 Features

- 📧 Email Phishing Detection
- 🌐 URL Analyzer
- 🛡️ Phishing Awareness Module
- 📚 Email Simulator
- 📊 Dashboard
- 💾 SQLite Database
- 📜 Email History
- 📄 Security Report

---

## 🛠 Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

---

## 📂 Project Structure

```
PhishGuard/
```

## ⚙️ Installation

```bash
git clone https://github.com/123DarshanM/PhishGuard.git

cd PhishGuard

pip install -r requirements.txt

python app.py
```
